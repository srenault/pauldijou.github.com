Some POCs

Hi there,

I’ve been a bit busy lately but I’m back on game. Looking for a job in Paris, I’ve been asked to do some code over more or less funny subjects and now that it’s done, I will share them, maybe it could help people to see some code around some technologies. It’s not at all perfect code, more experiments I’ve done.

Mowing is mainly a CoffeeScript project about giving orders to some automatic mowers and see the result. My goal was to test that new syntax around JavaScript and I have to admit it’s quite cool. You can see the result here (don’t over do it too large grids). The source code is here on GitHub

Mowing-java is the Java version of Mowing. So in fact, it’s nearly the same syntax since you can create class in CoffeeScript but this one runs on a JVM. See here for source code.

Finally, GSearch is an AngularJS application build using Yeoman. Its goal is to plug on GitHub API and allow you to search through repositories and users and display informations and stats about them. Here is the result and go there for source code.

Of course, any feedback is welcome! I can be bugs, features, questions, anything…

Quelques POCs

J’ai été un peu occupé ces derniers temps mais je suis de retour. Pendant ma recherche d’emploi sur Paris, on m’a demandé de réaliser quelques démos de code sur des sujets plus ou moins funs. Maintenant que c’est fait, je me suis dit que je pourrais les partager, peut être que cela pourrait intéresser quelques personnes de voir du code autour de certaines technologies. Ledit code est loin d’être parfait, il s’agit plus d’expériences que j’ai faites.

Mowing est principalement réalisé en CoffeeScript et son but est de donner des ordres à des tondeuses automatiques avant de les regarder tondre. L’intérêt était de tester cette nouvelle syntaxe autour de JavaScript et je dois bien avouer que j’ai été relativement séduit. Vous pouvez voir le résultat (n’allez pas tout casser avec des grilles trop larges). Le code source est ici, sur GitHub

Mowing-java est la version Java de *Mowing". Au final, c’est pratiquement la même syntaxe puisque vous pouvez créer des classes en CoffeeScript mais cette version tourne sur la JVM. Vous pouvez regarder ici pour le code source.

Enfin, GSearch est une application AngularJS organisée grâce à l’outil Yeoman. Son but est de se connecter aux API de GitHub afin de vous permettre d’effectuer des recherches sur les répertoires et les utilisateurs ainsi qu’afficher des informations et des statistiques à leur sujet. C’est ici pour le résultat et plutôt par là pour le code source.

Bien entendu, les retours sont les bienvenus, qu’il s’agisse de bugs, fonctionnalités, questions, ou n’importe quoi d’autre…
