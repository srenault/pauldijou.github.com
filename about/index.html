About the website

The core design is powered with Twitter Boostrap. Thanks to it, this website is fully responsive on any device and has a great look. There is also inspiration from Line25 for the title style.

The website code is generated by using Awestruct which is a real awesome tool. If you want a generator engine for HTML website and you are not afraid by a bit of Ruby code, you need to give it a try! Lots of inspiration from Arquillian website which is also build using Awestruct but with some many hand-made functionnalities.

The whole site is hosted at GitHub in a Git repo. The master branch contains the generated code you are currently seeing in your browser, the develop branch contains the Awestruct code in order to generate this code. For more info about hosting a website on GitHub , read GitHub Pages help.
